WarpTexture

Distorts one texture with another.

Parameters:

Target texture
This is the texture that you want to distort.
Warp texture
This is the texture that does the distorting. Each RGB component of the warp texture will distort the corresponding UVW/XYZ value of the Target texture (ie. the warp red will push the Target texture along U/X).

Note that the Warp texture values can go over one and less than zero by using the Output rollout in various textures. Negative values will push in the opposite direction of positive values.

Warp Scale
This is used to scale the values of the warp texture.

The "meaning" of this parameter changes depending on whether the Target texture is using UVW coordinates or XYZ coordinates.

  • When using UVW coordinates, the Warp Scale should probably be in the 0.0->1.0 range.
  • When using XYZ coordinates, you will probably have to set the Warp Scale to a much larger value since it now represents displacement in world (or object) space.

General Tips:

  • Heavily warped textures will be prone to aliasing. Use supersampling to work around this.
  • Render the sample scene as an animation.
